# Interactive Word Guessing Game

A Python word guessing game with two gameplay modes.

## Description

This game allows players to guess 5-letter words with hints about correct letters and their positions.

## Features

- **Solo Mode**: Guess a word randomly chosen by the computer
- **Two Player Mode**: Each player chooses a word for the other to guess
- Indication of correct letters and their positions
- Scoring system for multiplayer mode

## How to Play

1. Run the file `interactive_game.py`
2. Choose your game mode:
   - Mode 1: Solo (against the computer)
   - Mode 2: Two players
3. Follow the on-screen instructions

## Game Rules

- Words must contain exactly 5 letters
- You have 5 attempts to guess the word
- Correct letters in the right position are revealed
- You're informed if a letter is in the word but in the wrong position
